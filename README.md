## 文件说明
* config.py 是配置文件，用来填写个人信息
* save.py 是健康打卡脚本，根据配置文件中的信息进行健康打卡
* doSign.py 是晚签到脚本，位置已经写死（鲜橙大临潼校区）
* Utils.py 是工具类
* mylog.txt 是用来记录打卡信息的日志文件
